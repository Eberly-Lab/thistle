# Thistle
This repository contains the data and source code for the manuscript: Eberly JO, Hurd A, Oli D, Dyer AT, Seipel TF, Carr PM (2024) Compositional profiling of the rhizosphere microbiome of Canada thistle reveals consistent patterns across the United States northern Great Plains. Sci Rep 14: 18016. doi: 10.1038/s41598-024-69082-3
